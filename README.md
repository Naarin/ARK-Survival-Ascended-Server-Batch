# ARK-Survival-Ascended-Server-Batch

## Features

- Simple Setup: rename (exemple: The Island --> The_Island_WP) and execute.
- Backup Saves
- Download and Install SteamCMD
- Update Server
- Remove Useless Files (Movies ~5 Gb)
- Cores Allocation (2 Cores)
